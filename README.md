# Verdad
Verdad: An all-in-one platform for political awareness, offering comprehensive analyses, graphs, and data powered by artificial intelligence. It aims to promote rational voting and transparency by tracking the promises, positions, and actions of politicians and parties. Features include a political compass, profiles of politicians and parties, historical insights into key issues, and their societal impacts. An interactive chatbot ensures easy access to information, while users can engage by evaluating and following current topics.
